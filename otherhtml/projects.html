Y Combinator

A responsive corporate website for Y combinator, developed using HTML and CSS. The site features clean, modern design elements with an intuitive layout for easy navigation. It provides key information about the company services, products, and contact details, optimized for performance and accessibility across all devices

Standing Fan

This project is a fully animated standing fan created using only HTML and CSS, without any JavaScript. The goal was to demonstrate how far pure CSS can be pushed to create smooth, realistic animations and visually appealing UI elements.

Tv Console

This project showcases an interactive TV console scene I designed. It features two players on a football field, animated sky effects glowing hover interactions, and animated side speakers. The goal was to practice animation, visual effects, and interactive UI design.
